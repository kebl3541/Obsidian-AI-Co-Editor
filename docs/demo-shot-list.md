# Demo GIF shot list

A thirty second recording that sells the plugin. Record at 1280x800, light
theme, a clean vault with one essay note. Suggested tool: Kap or CleanShot,
exported as GIF under 8 MB for the README.

1. **Open on the note** (3 s). A paragraph of prose, the co-edit panel open
   on the right, status bar showing the plugin version.
2. **Ask in chat** (5 s). Type: "tighten the second sentence". Send. The
   "working" dot pulses.
3. **Ghosts appear** (6 s). The proposal renders in the text: struck words,
   green insertion, the small accept and reject buttons. Hover them briefly.
4. **Accept in text** (4 s). Click the check. The text settles; the accepted
   words glow in the collaborator's color.
5. **Select and ask** (7 s). Select a phrase, click the floating "Ask AI"
   button, type "make this more cautious", Send. New ghosts arrive.
6. **Reject this one** (3 s). Click the cross. The ghosts vanish, the text is
   untouched.
7. **Close on the panel** (2 s). Chat log with the exchange, "Their changes
   here" showing the accepted edit.

Caption for the store listing: "The machine proposes. You dispose."
